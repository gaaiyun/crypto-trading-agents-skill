## 🎯 功能说明

基于 CryptoTradingAgents 框架的 OpenClaw Skill，提供多 Agent 协同的加密货币交易分析。

### 核心功能
- 多 Agent 协同分析（市场、新闻、社交媒体、基本面）
- 多空辩论机制
- 支持 BTC/ETH/SOL 等币种
- 三种分析深度（quick/medium/deep）
- 多种输出格式（markdown/json/pdf）

### 数据源
- Binance API - 行情数据
- taapi.io - 技术指标
- CoinDesk - 新闻
- Reddit - 社交媒体情绪
- Fear & Greed Index

## 🚀 使用方法

### 1. 配置 API Key
```bash
cd 04-CryptoTradingAgents-Skill
copy references\.env.example references\.env
# 编辑 .env 文件，填入 API Key
```

### 2. 运行分析
```bash
# 分析 BTC
python scripts\run_analysis.py --ticker BTC

# 深度分析 ETH
python scripts\run_analysis.py --ticker ETH --depth deep

# JSON 输出
python scripts\run_analysis.py --ticker SOL --output json
```
